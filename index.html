Scorpinator

Purpose: To increase office efficiency using PestPac by providing custom modifications specific to our particular use-case. This can be done quite effectively through the use of Javascript user scripts.

To Install:

  1. Using Chrome, search for "Tampermonkey" extension and add to Chrome (For FireFox, search for "Greasemonkey" instead)
  2. Click on Tampermonkey icon in upper-right-hand corner. Go to "Dashboard"
  3. Click on the "+" icon to add a new script
  4. Navigate to http://RjHuffaker.github.io/scorpinator.js and copy the FULL contents to clipboard
  5. Paste text into new script and click the save button (looks like a disk)
  6. Make sure that Tampermonkey is enabled
  7. Check for updates to Scorpinator to get latest features :)

Features:

  • AutoGeocodinator: Automatically fetches missing geocodes in location/edit page. Highlights Save button to indicate that change has been made.
  • AutoProximinator: Provides a button that can be clicked to return the TWENTY nearest active service setups, color coded according to technician workload.
  • AutoContactinator: Disables phone links from initiating phone call to allow ZipWhip texting icon to function instead. Also redirects email link to use PestPac's built-in emailing application.
  • AutoDataFixinator: Automatically filters out periods from address fields in location/edit and bill-to/edit pages (allows for better searching of location addresses). Also copies primary phone number over to mobile number. Highlights save button to indicate that changes have been made.
  • AutoTaskinator: Creates task link on service order rows to assist with the creation of tasks. Clicking on an initial service will bring up AutoProximinator window with the ability to select service schedule.
  • TraverseAccountinator: Creates "Next" and "Prev" links near the Quick Search Input (only on the location/detail page) which allows you to jump immediately to the next or previous account ID.
  • ServiceOrderDuplicator: Adds a "Duplicate" button within every service order page, allowing you to click and create a carbon-copy of that service order.
  • AutoFollowUpinator: Creates link in applicable location task to create a new follow-up task 13 days out from the currently displayed task.
  • AutoSetupinator: Creates link in applicable location task to create a new service setup from the currently displayed task. Attempts to autofill necessary fields using data from task details.
  • AutoWelcomator: Creates link on location page to create a new welcome letter. Chooses the appropriate letter based on the division and attempts to input correct details based upon the first service setup.
  • AutoTextinator: Creates HeyMarket icon next to location and bill-to phone numbers. If clicked, starts a new message to that number in HeyMarket, but only if HeyMarket is already open. Also creates "Send Message" feature in any follow up task to send a text and complete the task.
  • More features coming soon!
